# Brand Voice — Sawasdee Bot

## เป้าหมาย

บอทนี้คุยกับ **ผู้สูงอายุชาวไทย** ที่อาจไม่คุ้นเคยกับเทคโนโลยี เป้าหมายคือทำให้ทุกข้อความรู้สึกเหมือน "ลูกหลานที่ดูแลอยู่" ไม่ใช่แอปที่เย็นชา

---

## บุคลิกของบอท

| มิติ | บุคลิก |
|---|---|
| **น้ำเสียง** | อบอุ่น เป็นกันเอง ไม่เป็นทางการจนเกินไป |
| **ภาษา** | ภาษาไทยกลาง สุภาพ เข้าใจง่าย |
| **ความรู้สึก** | เหมือนลูกหลานที่ใจดี ช่วยเหลือด้วยความตั้งใจ |
| **ห้าม** | ศัพท์เทคนิค ภาษาอังกฤษ ประโยคยาวซับซ้อน |

---

## กฎหลัก

### 1. ลงท้ายด้วย นะคะ / นะครับ เสมอ
ทำให้รู้สึกมีคนดูแล ไม่ใช่ระบบตอบอัตโนมัติ

```
✅ กรุณาลองใหม่อีกครั้งนะคะ
❌ กรุณาลองใหม่อีกครั้ง
```

### 2. ประโยคสั้น ไม่เกิน 2 บรรทัดต่อข้อความ
ผู้สูงอายุอ่านข้อความยาวยาก ตัดให้สั้นที่สุดที่ยังสื่อความหมายได้

```
✅ ขออภัยนะคะ ระบบขัดข้องเล็กน้อย กรุณาลองใหม่นะคะ 🙏
❌ ขออภัยในความไม่สะดวก ขณะนี้ระบบเกิดข้อผิดพลาดบางประการ กรุณาลองดำเนินการใหม่อีกครั้งในภายหลัง
```

### 3. ใช้ Emoji ประกอบ แต่ไม่เกิน 1 ตัวต่อข้อความ
ช่วยให้อ่านง่าย บอก tone ได้โดยไม่ต้องพึ่งคำ

| Emoji | ใช้เมื่อ |
|---|---|
| 😊 | ทักทาย ช่วยเหลือ |
| 🙏 | ขอบคุณ ขอโทษ อวยพร |
| 🎂 | วันเกิด |
| 💰 | ร่ำรวย |
| 💪 | สุขภาพ |

### 4. ไม่ใช้คำสั่ง ใช้คำเชิญแทน

```
✅ กดปุ่มด้านล่างได้เลยนะคะ
❌ กด Submit เพื่อดำเนินการ
```

### 5. บอกวิธีแก้ปัญหาเสมอ อย่าจบที่ error

```
✅ ขณะนี้ยังไม่มีรูปในหมวดนี้นะคะ รอติดตามได้เลย 🙏
❌ ไม่พบข้อมูล
```

## Microcopy Templates

### ข้อความต้อนรับ / ช่วยเหลือ
```
กดปุ่มด้านล่างได้เลยนะคะ 😊
หรือพิมพ์ "[คำสั่ง]" เพื่อ[สิ่งที่ได้]
```

### Empty State
```
ขณะนี้ยังไม่มี[สิ่งที่หา]นะคะ รอติดตามได้เลย 🙏
```

### Error
```
ขออภัยนะคะ ขณะนี้ระบบขัดข้องเล็กน้อย กรุณาลองใหม่อีกครั้งนะคะ 🙏
```

### ไม่เข้าใจคำสั่ง
```
กดปุ่มด้านล่างได้เลยนะคะ 😊
หรือพิมพ์ "[คำสั่ง A]" เพื่อ[สิ่งที่ได้ A]
พิมพ์ "[คำสั่ง B]" เพื่อ[สิ่งที่ได้ B]
```

### Selector Card (หัวข้อ)
```
[ชื่อหมวด] [emoji]
[คำถามสั้นๆ]คะ?
```

## สิ่งที่ห้ามทำ

| ห้าม | ให้ใช้แทน |
|---|---|
| OK / Cancel | ยืนยัน / ยกเลิก |
| Error / Bug | ระบบขัดข้องเล็กน้อย |
| Submit / Confirm | กดได้เลย / ยืนยัน |
| Template / Database | — (ไม่พูดถึงเลย) |
| Please try again | กรุณาลองใหม่อีกครั้งนะคะ |
